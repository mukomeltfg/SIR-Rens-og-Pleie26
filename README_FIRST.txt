SIR Rens & Pleie — готовый комплект

1. Распакуйте ZIP.
2. Загрузите ВСЕ файлы из этой папки в корень GitHub-репозитория
   SIR-Rens-og-Pleie1 с заменой одноимённых файлов.
3. Нажмите Commit changes.
4. Подождите 2–5 минут.
5. Откройте:
   https://mukomeltfg.github.io/SIR-Rens-og-Pleie1/admin.html
6. Нажмите Ctrl+Shift+R.

Что исправлено:
- новый адрес сайта;
- новый Supabase Project URL;
- Publishable key из вашего проекта;
- вход владельца;
- кнопка «Забыли пароль»;
- установка нового пароля после письма восстановления;
- загрузка заказов;
- изменение статуса;
- вызов Edge Function complete-order;
- принудительное обновление файлов без старого кэша.

Пароль не хранится в файлах сайта. Его хранит Supabase.
